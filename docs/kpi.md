# Business Questions & KPI Design

## Project Goal

- This project 

## Business Questions
1. What skills are most demanded for Data Analysts?
2. Which skills correlate with higher salaries?
3. Does remote work affect pay?
4. Which countries have highest demand and pay?
5. Do degree requirements affect accessibility and salary?
6. How is demand changing over time?

## KPIs

### Market Overview
- Total Data Analyst Jobs
- % Remote Roles
- Average Salary
- % Jobs Without Degree Requirement

### Skill Demand
- % Jobs Requiring SQL
- % Requiring Python
- % Requiring Excel
- % Requiring Tableau
- % Requiring R

### Compensation
- Avg Salary by Skill
- Avg Salary: Remote vs On-site
- Avg Salary by Country

### Accessibility
- % No-Degree Jobs
- Salary: Degree vs No Degree

### Trends
- Jobs per Month
- Skill Demand Over Time
